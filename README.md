#2-Wheeler Battery Simulation
The 2-Wheeler Battery Simulation project is designed to predict the health of a 2-wheeler battery based on input parameters such as cycles, temperature, state of charge (SoC), average kilometers driven per day, and age in months.


#Features
Prediction Form: Allows users to input parameters related to the battery's usage.
Health Prediction: Predicts the health of the battery based on the provided inputs.
Web Interface: Simple and intuitive web-based interface for ease of use.
#Usage
Input Form:
Fill out the form with the following parameters:
Cycles: Number of charge-discharge cycles.
Temperature: Operating temperature of the battery.
State of Charge (SoC): Percentage of charge remaining in the battery.
Average Kilometers Driven per Day: Estimated average distance driven per day.
Months Old: Age of the battery in months.
Prediction:
Click on the "Predict Battery Health" button to obtain the predicted battery health based on the input parameters.
